Workspace Roles

Workplace Admin (WA)
  • manage the workplace settings, manually invite people, handle payments etc
  • restore or archive groups
  • create private groups
  • access the global dashboard
  • assign roles to other members
WORK Managers - WM
  • create private groups
  • manage users and roles inside the groups they create
  • access the global dashboard
  • delete their groups
Group Managers - GM
  • they are created by the WM, inside a group
  • manage users and roles inside the group they belong
  • can access the group dashboard
  • can't create other groups
  • can delete their groups
Members - M
  • they can create AGORAs
  • can create content inside the groups
Guests - G
  • isolated to the Group they are invited
  • can't access any Agora(Public Groups)
